CLEO Redux

CLEO Redux is the official port of CLEO Library for re3 and reVC . It brings a joy of enhancing the gameplay with countless mini-scripts that are easy to add and remove. The new version aims to address decade-long issues and improve the developer experience too.

Find Out More

Features

Easy to install

Ships as a single binary file (  cleo.asi  ) that you need to copy to the game directory. Uninstalling is as easy as deleting the cleo.asi file. Works with Windows D3D9 MSS 32bit version of re3 and reVC (any* build).

Supports CS scripts

Similar to other CLEO implementations CLEO Redux loads scripts from the CLEO directory and runs them as you start playing. The majority of CLEO scrips made for GTA III and Vice City should work with the new version**.

Runs its own runtime

CLEO Redux can interpret and execute text scripts written in the language conforming to the ECMAScript 5.1 standard***

And more

Visit the repository on GitHub to learn about other cool features

* As long as there is the file re3.pdb (for re3 ) or reVC.pdb (for reVC ). Due to the dynamic nature of memory addresses in those implementations CLEO Redux relies on debug information stored in the PDB file to correctly locate itself.

** At the moment CLEO Redux does not support most of the common CLEO commands yet. If the script only uses vanilla commands, or commands listed here it should work.

*** Simply speaking, CLEO Redux can run JavaScript. This feature is highly experimental and little to no bindings have been implemented. You can find an example "Hello, world" script in the repository on GitHub.
